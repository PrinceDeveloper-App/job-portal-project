# Job Portal Application

## Description
A web-based Job Portal System that enables employers to post job listings and publish them across multiple countries, while allowing job seekers to search and apply for jobs via WhatsApp and email. The platform helps streamline recruitment by providing an easy interface for managing job postings and applications.

## Features
- Employer dashboard to post/manage jobs
-Job search and filtering
-Job details page
- Admin job management
- Job view tracking
- PDF job report download
-Responsive UI
- Stripe payment integration for premium listings

## Technologies Used

- Backend: PHP / CodeIgniter
- Frontend: HTML, CSS, JavaScript,
- jQuery
- Database: MySQL
- Version Control: Git & GitHub
- Other Tools: DomPDF, REST APIs
